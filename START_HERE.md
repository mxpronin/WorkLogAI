# WorkLog AI — старт для следующего агента

Это Android-first приложение на HTML, CSS и JavaScript, упакованное через Capacitor 7. Веб-страница нужна только для локальной проверки общих исходников.

## С чего начать

1. Прочитать `AGENTS.md`, затем `docs/HANDOFF.md`, `README.md` и `docs/NEW_CHAT_PROMPT.md`.
2. Сохранить совместимость IndexedDB и не изменять фактические `note` и `actualMinutes` при работе со СППР.
3. После изменения кода приложения увеличить версию и Android `versionCode`, выполнить `node --check`, `npm test`, мобильную визуальную проверку и `npm run android:apk`.

## Текущее состояние

- Версия: `0.6.15`, Android `versionCode 22`.
- Главный экран — календарь.
- СППР-текст хранится отдельно в `spprDescription`; его можно редактировать без изменения фактической записи.
- Лимит СППР задачи является накопительным и необязательным. Общий лимит массово применяется через настройки, но отдельную задачу можно скорректировать вручную.
- Последний APK: `android/app/build/outputs/apk/debug/WorkLog-AI-0.6.15.apk`.

Полная история решений, ограничений и последней проверки — в `docs/HANDOFF.md`.
